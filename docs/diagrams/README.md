# Diagramas do Modelo Relacional

## Arquivo principal

- `fivestars_erd.dbml`

## Como gerar visual (dbdiagram)

1. Abra [dbdiagram.io](https://dbdiagram.io).
2. Crie um novo diagrama.
3. Cole o conteúdo de `docs/diagrams/fivestars_erd.dbml`.
4. Ajuste o layout arrastando as tabelas.
5. Exporte em PNG/SVG/PDF.

## Como usar no Draw.io (diagrams.net)

1. Exporte em `SVG` no dbdiagram.
2. Abra [draw.io](https://app.diagrams.net/).
3. Importe o SVG e aplique o tema visual da apresentação (cores, títulos, fundo).

Isso mantém a modelagem correta e facilita chegar no estilo visual do exemplo.
